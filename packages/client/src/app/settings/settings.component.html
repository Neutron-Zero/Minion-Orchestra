How to Monitor Claude Code

Quick Start

  1. Start Minion Orchestra: npm start (starts both server and UI)
  2. Run Claude with monitoring: ./claude-simple
  3. Watch real-time activity in the dashboard!

Available Scripts

  • ./claude-simple - Interactive Claude with basic monitoring
  • ./claude-with-monitor.sh "prompt" - Detailed monitoring for specific tasks
  • ./start-monitoring.sh - Monitor existing Claude output via pipe

What Gets Monitored

  • Agent registration and status
  • Tool usage (Read, Write, Edit, Bash, etc.)
  • File operations and commands
  • Real-time logs and metrics
  • Task progress and completion
