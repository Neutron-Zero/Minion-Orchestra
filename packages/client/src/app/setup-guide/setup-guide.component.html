Getting Started with Minion Orchestra

1. Start Minion Orchestra

Run this command in the minion-orchestra folder:

npm start

This automatically starts:

  • WebSocket server (port 3000)
  • Auto-monitor daemon (detects Claude processes)
  • Angular UI (port 4201)
 2. Run Claude Code

Start Claude Code normally in any terminal:

claude

The auto-monitor will detect it and show it as a persistent agent.
